# Setup

1. Create a public GitHub repository named `Tinh0804`.
2. Copy `README.md`, `assets/`, and `.github/` into the repository root.
3. Push the files to the `main` branch.
4. The profile works immediately without the optional snake animation.
5. To generate contribution-snake assets later, open **Actions** and run **Generate contribution snake** manually.

The local SVG artwork is self-contained, so no separate image hosting is required.
